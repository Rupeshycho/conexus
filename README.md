# Conexus
<h1>Social Media App and Communication</h1> 
- **Overview** <br>
Conexus is a Flutter-based social media application where users can connect, share posts, and interact in real-time.

- **Features**  <br>
<pre>
User authentication (Firebase)
Create and view posts
Like and comment system
Real-time chat
Home Feed Search Posts, people, connections, & more. 
</pre>

- **API/Backend Flow** <br>
User => App => Firebase => Database => Response => UI 

-**Tech Stack** <br>
<pre>
Flutter (Frontend)
  Dart Programming
Firebase (Backend)
Database(MySQL)
UI/UX(Figma, Canva)
</pre>
